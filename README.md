# Capitalis | Gestão de Patrimônio de Elite

![Capitalis Preview](./public/favicon.png)

> Estratégias financeiras de elite para investidores que exigem solidez, transparência e performance.

## Sobre o Projeto

Capitalis é uma landing page de alta conversão desenvolvida para uma gestora de patrimônio fictícia (ou template). O projeto foca em transmitir autoridade, exclusividade e confiança através de um design "Private Bank" sofisticado.

A aplicação inclui:
- **Design System Premium:** Paleta de cores exclusiva (Navy & Gold).
- **Simulador de Investimentos:** Ferramenta interativa com "Labor Illusion" e inputs precisos.
- **Gráficos de Performance:** Visualização de dados com Recharts.
- **Animações Fluidas:** Uso de Framer Motion para micro-interações e transições.

## Tecnologias

- **Framework:** React + Vite
- **Estilização:** Tailwind CSS v3
- **Componentes:** Shadcn/UI + Radix UI
- **Animações:** Framer Motion
- **Ícones:** Lucide React
- **Gráficos:** Recharts

## Como Rodar

```bash
# Instalar dependências
npm install

# Rodar servidor de desenvolvimento
npm run dev

# Build para produção
npm run build
```
